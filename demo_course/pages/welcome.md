# Welcome!

In this demo course you will find quiz, assignment, and page examples.

Some assignments and pages have been made with Jinja templates, which allow for dynamic content generation.

While other assignments and pages are static files.

You will also find a quiz with all current question types.

Feel free to modify this course to suit your needs and look at the code to see how it works, 
which can be found here: [demo_course](https://github.com/BYU-CS-Course-Ops/mdx-canvas/tree/updated-readme/demo_course)

If you need help with further customization, please refer to the [documentation](https://github.com/BYU-CS-Course-Ops/mdx-canvas/)

Have fun!
