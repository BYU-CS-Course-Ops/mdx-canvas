# MDXCanvas Demo Course

Welcome to the demo course for MDXCanvas! This course demonstrates how you can build and deploy Canvas content programmatically.

---

## Course Description

This course explores the use of MDXCanvas to automate Canvas course content creation using Markdown, XML, and Jinja templates. It is intended as a showcase for developers and instructors integrating with Canvas LMS.

## Learning Outcomes

By the end of this demo course, you will be able to:

- Structure and deploy Canvas content with MDXCanvas.
- Use advanced tags such as `<include>`, `<quiz>`, `<assignment>`, and `<module>`.
- Apply Jinja templating to scale course content.
- Customize Canvas appearance using external CSS.
