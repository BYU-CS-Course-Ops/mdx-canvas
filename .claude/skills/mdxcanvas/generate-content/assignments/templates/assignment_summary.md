# Assignment Summary Template

After generating an assignment, display a summary using this format:

```
**Assignment Generated:**

| Detail        | Value                  |
|---------------|------------------------|
| **Title**     | [assignment title]     |
| **File**      | [file path]            |
| **Points**    | [points_possible]      |
| **Due**       | [due_at or "Not set"]  |
| **Group**     | [assignment_group]     |
```

## Example

**Assignment Generated:**

| Detail     | Value                                        |
|------------|----------------------------------------------|
| **Title**  | Homework 1: Variables and Data Types         |
| **File**   | content/assignments/homework-1.canvas.md.xml |
| **Points** | 100                                          |
| **Due**    | Jan 15, 2025, 11:59 PM                       |
| **Group**  | Homework                                     |
