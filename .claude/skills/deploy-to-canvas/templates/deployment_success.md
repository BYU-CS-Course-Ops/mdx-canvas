# Deployment Success Template

After a successful deployment, ALWAYS display a summary using this exact format:

```
**Deployment Successful!**

| Detail           | Value                  |
|------------------|------------------------|
| **Content Type** | [Quiz/Assignment/Page] |
| **Name**         | [Content title]        |
| **Course**       | [Course name]          |
| **Link**         | [Canvas URL]           |
```
