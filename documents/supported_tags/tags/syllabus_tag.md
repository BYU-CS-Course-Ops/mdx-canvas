# `<syllabus>` Tag

The `<syllabus>` tag defines the course syllabus content. When included, it is automatically published to the course's **Syllabus** section in Canvas.

You can use Markdown, HTML, or plain text inside the tag body to structure your syllabus content.

## Example

```xml
<syllabus>
    # Course Syllabus

    Welcome to the course! Please review the information below.

    ## Course Description

    - Assignments: 40%
    - Quizzes: 30%
    - Final Project: 30%

    ## Policies

    Late submissions are accepted up to 48 hours with a penalty.
</syllabus>
```
