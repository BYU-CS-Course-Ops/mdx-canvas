Use the sub-skills in this directory when working with MDXCanvas. Available skills:

- **content_design** — design content before writing (inputs, structure, decisions)
- **content_types** — overview of all Canvas resource types; sub-skills for assignments, modules, pages, and quizzes (
  including all 12 question types)
- **course_structure** — directory layout, entry points, course_info, global_args, scaffolding; sub-skills for
  understanding existing course codebases and organization patterns (days, lectures, modules, units)
- **jinja** — Jinja2 templating: variables, macros, loops, filters, MDXCanvas-injected functions, args files
- **special_tags** — infrastructure tags: `<include>`, `<file>`, `<img>`, `<zip>`, `<course-link>`, `<course-settings>`,
  `<timestamp/>`
