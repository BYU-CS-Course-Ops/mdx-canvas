# MDXCanvas Reference

Use this hub to navigate syntax and attribute reference for MDXCanvas. Select the sub-skill that matches your lookup.

## Sub-Skills

| Sub-skill                        | What it covers                                                                                                     |
|----------------------------------|--------------------------------------------------------------------------------------------------------------------|
| `course-setup/skill.md`          | Disk layout, file extensions, naming conventions, `course_info`, `global_args`                                     |
| `canvas-tags/skill.md`           | All Canvas resource tags: `<assignment>`, `<quiz>`, `<page>`, `<module>`, `<announcement>`, `<syllabus>`           |
| `helper-tags/skill.md`           | Infrastructure tags: `<include>`, `<file>`, `<img>`, `<zip>`, `<course-link>`, `<course-settings>`, `<timestamp/>` |
| `jinja/skill.md`                 | Jinja2 templating syntax, args files, MDXCanvas-injected functions                                                 |
| `organization-patterns/skill.md` | How courses are organized: days, lectures, modules, units                                                          |

## Routing

- Looking up a tag attribute or syntax → read `canvas-tags/skill.md` or `helper-tags/skill.md`
- Setting up a new course on disk → read `course-setup/skill.md`
- Writing a Jinja template → read `jinja/skill.md`
- Understanding how a course is structured → read `organization-patterns/skill.md`
